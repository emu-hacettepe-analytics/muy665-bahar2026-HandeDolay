library(tidyverse)
library(readxl)
library(scales)
library(broom)
library(knitr)

veri <- read_excel("fakulte_cinsiyet_verisi.csv.xlsx")

names(veri) <- c("Yil", "Cinsiyet", "Fakulte", "Kisi_Sayisi")

veri <- veri %>%
  mutate(
    Yil = as.character(Yil),
    Cinsiyet = as.character(Cinsiyet),
    Fakulte = as.character(Fakulte),
    Kisi_Sayisi = as.numeric(Kisi_Sayisi),
    Yil = factor(
      Yil,
      levels = c("2020-2021", "2021-2022", "2022-2023", "2023-2024", "2024-2025")
    ),
    Cinsiyet = factor(
      Cinsiyet,
      levels = c("ERKEK", "KADIN", "TOPLAM")
    )
  )

veri_cinsiyet <- veri %>%
  filter(Cinsiyet %in% c("ERKEK", "KADIN"))

veri_toplam <- veri %>%
  filter(Cinsiyet == "TOPLAM")

veri_genis <- veri %>%
  pivot_wider(
    names_from = Cinsiyet,
    values_from = Kisi_Sayisi
  ) %>%
  rename(
    Erkek = ERKEK,
    Kadin = KADIN,
    Toplam = TOPLAM
  )

oran_veri <- veri_genis %>%
  mutate(
    Kadin_Orani = Kadin / Toplam * 100,
    Erkek_Orani = Erkek / Toplam * 100
  )

yillik_toplam <- veri_toplam %>%
  group_by(Yil) %>%
  summarise(
    Toplam_Ogrenci = sum(Kisi_Sayisi),
    .groups = "drop"
  )

fakulte_toplam <- veri_toplam %>%
  group_by(Fakulte) %>%
  summarise(
    Toplam_Ogrenci = sum(Kisi_Sayisi),
    Ortalama_Ogrenci = mean(Kisi_Sayisi),
    .groups = "drop"
  ) %>%
  arrange(desc(Toplam_Ogrenci))

cinsiyet_toplam <- veri_cinsiyet %>%
  group_by(Cinsiyet) %>%
  summarise(
    Toplam_Sayi = sum(Kisi_Sayisi),
    .groups = "drop"
  )

yil_cinsiyet <- veri_cinsiyet %>%
  group_by(Yil, Cinsiyet) %>%
  summarise(
    Toplam = sum(Kisi_Sayisi),
    .groups = "drop"
  )

fakulte_cinsiyet <- veri_cinsiyet %>%
  group_by(Fakulte, Cinsiyet) %>%
  summarise(
    Toplam = sum(Kisi_Sayisi),
    .groups = "drop"
  )

Proje Genel Bakış ve Kapsamı

Bu projenin amacı, seçilmiş fakültelerde öğrenim gören kadın ve erkek öğrenci sayılarını yıllara göre incelemek ve fakülte bazında cinsiyet dağılımındaki değişimleri analiz etmektir. Çalışmada Eğitim Fakültesi, İktisadi ve İdari Bilimler Fakültesi, Mühendislik Fakültesi, Sağlık Fakültesi ve Teknoloji Fakültesi ele alınmıştır.

Analiz kapsamında 2020-2021 ile 2024-2025 akademik yılları arasındaki öğrenci sayıları kullanılmıştır. Veri setinde her fakülte için erkek öğrenci sayısı, kadın öğrenci sayısı ve toplam öğrenci sayısı yer almaktadır.

Bu çalışmada temel amaç, fakülteler arasında öğrenci yoğunluğunu, kadın-erkek dağılımını ve yıllara göre değişimi görselleştirerek yorumlamaktır. Böylece yükseköğretimde farklı fakülte türlerine göre kadın ve erkek öğrencilerin temsil düzeyleri hakkında genel bir değerlendirme yapılması hedeflenmiştir.

Veri

Bu projede kullanılan veri seti, fakültelere ve akademik yıllara göre kadın ve erkek öğrenci sayılarını içermektedir. Veri setinde beş farklı fakülte ve beş akademik yıl bulunmaktadır. Her satır, belirli bir akademik yılda belirli bir fakülteye ait öğrenci sayılarını göstermektedir.

Veri setindeki temel değişkenler şu şekildedir:

- **Yil:** Akademik yılı göstermektedir.
- **Fakulte:** Öğrenci sayısının ait olduğu fakülteyi belirtmektedir.
- **Erkek:** İlgili fakültedeki erkek öğrenci sayısını göstermektedir.
- **Kadin:** İlgili fakültedeki kadın öğrenci sayısını göstermektedir.
- **Toplam:** Kadın ve erkek öğrenci sayılarının toplamını göstermektedir.

Bu çalışmada veri seti, R ortamına `readr` kütüphanesinin `read_csv()` fonksiyonu kullanılarak klasörde yer alan CSV dosyasından aktarılmıştır.

Tercih Sebebi

Bu veri setinin tercih edilme nedeni, yükseköğretimde fakültelere göre kadın ve erkek öğrenci dağılımını incelemeye imkân sağlamasıdır. Kadın ve erkek öğrencilerin farklı fakültelerdeki temsil oranları, eğitim alanındaki yönelimleri ve akademik tercihleri anlamak açısından önemlidir.

Özellikle Mühendislik ve Teknoloji fakültelerinde erkek öğrenci oranının daha yüksek olması beklenirken, Eğitim ve Sağlık fakültelerinde kadın öğrenci oranının daha yüksek olabileceği düşünülmektedir. Bu nedenle veri seti, fakülteler arasında cinsiyet temelli farklılıkları gözlemlemek için uygun bir yapı sunmaktadır.

Ayrıca veri setinin yıllara göre düzenlenmiş olması, yalnızca mevcut dağılımı değil, zaman içerisindeki değişimi de incelemeyi mümkün kılmaktadır. Bu yönüyle veri seti hem fakülteler arası karşılaştırma hem de yıllara göre değişim analizi yapmak için uygundur.

Model Uydurma

Bu projede temel amaç keşifsel veri analizi yapmak olsa da, yıllara göre toplam öğrenci sayısındaki eğilimi incelemek için basit doğrusal model kurulmuştur. Modelde bağımlı değişken toplam öğrenci sayısıdır. Bağımsız değişken ise akademik yılın sayısal karşılığıdır.

Model genel olarak şu şekilde ifade edilebilir:

**Toplam Öğrenci Sayısı = β0 + β1 × Yıl + ε**

## Model Uydurma

Bu projede temel amaç keşifsel veri analizi yapmak olsa da, yıllara göre toplam öğrenci sayısındaki eğilimi incelemek için basit doğrusal model kurulmuştur. Modelde bağımlı değişken toplam öğrenci sayısıdır. Bağımsız değişken ise akademik yılın sayısal karşılığıdır.\
\
Model genel olarak şu şekilde ifade edilebilir:\
\
\*\*Toplam Öğrenci Sayısı = β0 + β1 × Yıl + ε\*\*\
\
Burada β1 katsayısı, yıllara göre toplam öğrenci sayısının artış veya azalış eğilimini göstermektedir. Katsayının pozitif olması artış eğilimini, negatif olması ise azalış eğilimini ifade eder.\
\

```{r}
#| label: model-verisi

model_veri <- veri_toplam %>%
  mutate(
    Yil_Sayi = as.numeric(Yil)
  )

model_veri %>%
  select(Yil, Yil_Sayi, Fakulte, Kisi_Sayisi) %>%
  head() %>%
  kable()
```

Her fakülte için ayrı doğrusal model kurulmuştur.

```{r}
#| label: model-sonuclari

model_sonuclari <- model_veri %>%
  group_by(Fakulte) %>%
  do(tidy(lm(Kisi_Sayisi ~ Yil_Sayi, data = .)))

model_sonuclari %>%
  kable()
```

Model sonuçlarının daha anlaşılır olması için yalnızca eğilim katsayıları aşağıda gösterilmiştir.

```{r}

egilim_sonuclari <- model_sonuclari %>%

  rename(
    Egilim_Katsayisi = estimate,
    Standart_Hata = std.error,

    P_Degeri = p.value

egilim_sonuclari %>%
```

Model sonuçlarına göre eğilim katsayısı negatif olan fakültelerde toplam öğrenci sayısının yıllar içinde azalma eğiliminde olduğu, pozitif olan fakültelerde ise artış eğilimi olduğu söylenebilir. Ancak veri seti yalnızca beş akademik yılı kapsadığı için bu modeller güçlü bir tahmin modeli olarak değil, genel eğilimi gösteren destekleyici analizler olarak değerlendirilmelidir.\
